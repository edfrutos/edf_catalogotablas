Ayuda - Scripts de Base de Datos
¿Qué son estos scripts?

Estos scripts son herramientas de mantenimiento para la base de datos de la aplicación. Permiten realizar tareas como:

  • Corregir inconsistencias en los datos
  • Generar informes
  • Realizar copias de seguridad
  • Migrar o transformar datos
  • Otras tareas administrativas
Recomendaciones de seguridad
¡Precaución! Los scripts pueden modificar datos. Asegúrate de entender qué hace un script antes de ejecutarlo.
¿Cómo usar?
  1. Selecciona un script de la lista
  2. Opcionalmente, añade argumentos si el script los requiere
  3. Haz clic en "Ejecutar" para ejecutar el script con los argumentos proporcionados
  4. Usa el botón "Ayuda" para ver la documentación de un script
Solución de problemas

Si un script falla:

  • Verifica que los argumentos sean correctos
  • Consulta los mensajes de error en la sección de resultados
  • Revisa los logs del servidor para más detalles
  • Si el problema persiste, contacta al administrador del sistema
