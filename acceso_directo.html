Acceso Directo al Sistema

Información: Esta página proporciona acceso directo al sistema de administración sin pasar por el proceso normal de autenticación.
Advertencia: Esta página es para uso exclusivo del administrador del sistema y debe ser eliminada después de resolver los problemas de acceso.

Diagnóstico del Sistema

  • La aplicación está siendo servida por Gunicorn a través de un socket Unix.
  • La configuración del servidor está controlada por systemd (servicio edefrutos2025).
  • Las credenciales de MongoDB están configuradas en /etc/default/edefrutos2025.
  • El punto de entrada WSGI es wsgi:app.
Soluciones si no funciona el acceso:
  1. Ejecutar python3 repair_admin.py para restaurar el usuario administrador.
  2. Reiniciar los servicios: sudo systemctl restart edefrutos2025 && sudo systemctl restart apache2
  3. Verificar los logs: tail -n 100 logs/gunicorn_error.log
