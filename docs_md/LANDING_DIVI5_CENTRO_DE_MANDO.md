## 1. El centro de mando

El Centro de Mando de Divi 5 convierte acciones repetitivas en comandos rápidos.

Si ya piensas en pasos como:

> "Añadir un módulo de descripción, añadir un grupo dentro, añadir dos botones, aplicar mi preajuste principal..."

ahora puedes resolverlo escribiendo una sola secuencia.

No se trata solo de crear más rápido. También puedes:

- navegar entre páginas en segundos,
- abrir modales sin recorrer menús,
- saltar directo a grupos de opciones,
- ejecutar acciones de edición comunes sin depender del ratón.

### Beneficios clave

- **Velocidad real:** menos clics, más resultados.
- **Flujo continuo:** sin interrupciones entre acciones.
- **Precisión:** ejecutas exactamente lo que necesitas.
- **Escalabilidad:** ideal para usuarios avanzados y equipos.

### 1.1 Categorías de comandos

#### Elemento

Añade cualquier elemento a la página.  
También permite estructuras anidadas y múltiples elementos en una misma secuencia.

Ejemplo:

```text
blurb > group > button *2
```

Resultado: un módulo de blurb, un grupo dentro y dos botones dentro del grupo.

#### Modal

Abre ventanas del constructor sin buscar en menús.

- Escribe `variables` para abrir gestión de variables.
- Escribe `exportar` para ir directo a importar/exportar.

#### Configuración

Navega a grupos de opciones dentro de un panel.

Ejemplo:

- Escribe `animación` y entra directamente al bloque correspondiente.

#### Ver

Cambia entre modos de visualización: puntos de interrupción, wireframe y más.

#### Modificar

Acciones del menú contextual también disponibles por comando:

- duplicar,
- eliminar,
- copiar,
- cortar,
- pegar,
- restablecer.

#### Navegar

Cambia de página o de área administrativa sin fricción.

- Escribe el nombre de una página y entra a editarla.
- Salta a Creador de Temas u Opciones del Tema por nombre.

#### Preajuste

Aplica preajustes sin recorrer listas largas.

Ejemplo de cola de preajustes:

```text
Preajuste de descripción (Cmd+Intro)
Preajuste de borde (Cmd+Intro)
Preajuste de espaciado (Intro)
```

#### Página

Comandos de nivel de página:

- guardar,
- salir,
- vista previa.

## 2. Operadores de comando

El Centro de Mando no se limita a ejecutar un solo comando.
Puedes encadenar acciones, controlar estructuras y filtrar resultados mientras escribes.

### 2.1 Comandos de cola

Para encolar comandos de **Elemento** y **Preajuste**:

1. ejecuta con `Cmd + Intro` para enviar a cola,
2. repite para añadir más comandos,
3. pulsa `Intro` para ejecutar toda la cola.

Esto te permite construir una secuencia completa en un solo flujo.

### 2.2 Operadores de cola de comandos

#### `>`

Coloca un elemento dentro del elemento anterior en cola.

Ejemplo:

```text
blurb > button
```

Añade un botón dentro del blurb.

#### `*`

Multiplica elementos en la cola.

Ejemplo:

```text
blurb > button *2
```

Añade un blurb con dos botones.

#### `^`

Coloca un elemento encima del elemento editado.

Por defecto, los elementos se añaden debajo.  
Con `^` y `>` controlas si se insertan arriba o dentro.

### 2.3 Combine ajustes preestablecidos y elementos en una sola cola

Puedes mezclar creación de elementos y aplicación de preajustes en la misma cadena.

Ejemplo:

```text
Descripción > Grupo > Botón + Ajuste preestablecido principal *2
```

Resultado:

- crea un grupo dentro de una descripción,
- crea dos botones dentro del grupo,
- aplica el preajuste principal a ambos botones.

### 2.4 Filtrar sobre la marcha

Cuando hay muchos comandos, usa el operador `:` para acotar por categoría mientras escribes.

Ejemplo:

- Escribe `modal` y el sistema prioriza comandos de tipo modal.

Esto reduce ruido y acelera la selección del comando correcto.

## FAQ rápida

### ¿Necesito ser un usuario avanzado para usar el Centro de Mando?
No. Puedes empezar con comandos simples y, cuando quieras, pasar a colas y operadores.

### ¿Qué gano al usar comandos en lugar de clics?
Velocidad, menos fricción y mayor consistencia al repetir tareas frecuentes.

### ¿Puedo combinar estructura y estilo en una sola acción?
Sí. Puedes crear elementos y aplicar preajustes en la misma cola.

### ¿El Centro de Mando sirve también para navegar?
Sí. Permite cambiar de páginas y abrir áreas clave del sitio al instante.
